# Saksham Garg - Portfolio Website

Personal portfolio site built with React + Vite.

## Project Structure

```
saksham-art-portfolio/
├── src/
│   ├── components/        # Reusable components (Header, Footer, etc.)
│   ├── pages/             # Page components (Home, Film, Poetry, etc.)
│   ├── data/              # Data files for content
│   ├── App.jsx            # Main app with routing
│   ├── main.jsx           # Entry point
│   └── index.css          # All styles
├── public/
│   └── images/            # All images
└── package.json
```

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit http://localhost:5173

## Build for Production

```bash
npm run build
```

The `dist/` folder will contain the static files ready to deploy.

## Deployment

- Vercel (automatic with Git push)
- Netlify (drag & drop the `dist` folder)
- GitHub Pages
- Any static hosting service
